# 🌙 ArVim (المحرر العربي الحديث)

**ArVim** هو مشروع مفتوح المصدر صُمم ليجعل من محرر النصوص الأسطوري **Vim** أداة مثالية للمبرمج والكاتب العربي في عام 2026.

## ✨ المميزات الحالية:
* **دعم كامل للغة العربية (RTL):** تهيئة ذكية للكتابة من اليمين إلى اليسار.
* **خرائط الأوامر العربية (Arabic Mapping):** تحكم في Vim بأوامر عربية دون الحاجة لتغيير لغة لوحة المفاتيح.
* **أرقام الأسطر النسبية:** تنقل بسرعة البرق بين الأسطر البرمجية.
* **شريط حالة معرب:** واجهة معلوماتية تظهر لك حالة الملف واللغة باللغة العربية.
* **تصميم داكن (Dark Mode):** مريح للعين ومناسب لجلسات العمل الطويلة.

## 🚀 كيف تبدأ؟
لتركيب الإعدادات، أضف السطر التالي لملف `.vimrc` الخاص بك:
\`\`\`vim
source ~/ArVim_Project/core/arvim.vim
\`\`\`
