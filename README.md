# ArVim (المحرر العربي الحديث)

**ArVim** هو مشروع يهدف لتطوير نسخة مخصصة من محرر النصوص الشهير Vim لتناسب المبرمجين والكتاب العرب في عام 2026.

## المميزات الحالية:
- دعم كامل للغة العربية (RTL).
- أوامر Vim تعمل باللغة العربية (Langmap).
- متوافق مع خطوط Nerd Fonts الحديثة.

## كيفية الاستخدام:
قم بإضافة `source ~/ArVim_Project/core/arvim.vim` إلى ملف `.vimrc` الخاص بك.
