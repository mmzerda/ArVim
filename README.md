## ✨ المميزات الرئيسية (Features)
مشروع **ArVim** صُمم ليوفر لك تجربة كتابة وبرمجة عربية "سلسة" دون تعقيد:

1.  **دعم RTL التلقائي:** كتابة وتشكيل الحروف العربية من اليمين إلى اليسار بذكاء.
2.  **خريطة الأوامر (Langmap):** تحكم في Vim بالأحرف العربية (مثلاً `حب` للحفظ و `خر` للخروج) دون تبديل اللغة.
3.  **التدقيق الإملائي (F3):** اكتشاف وتصحيح الأخطاء الإملائية العربية بضغطة زر.
4.  **التنقل النسبي:** أرقام أسطر ذكية تجعلك تقفز بين الأسطر بسرعة البرق.
5.  **الوضع الداكن (Dark Mode):** سمة بصرية مريحة للعين، متوافقة تماماً مع طرفيات Linux و antiX.

## 🛠️ دليل الاستخدام (Quick Guide)

### ⌨️ الاختصارات السحرية (Snippets)
أثناء وضع الكتابة (Insert Mode)، اكتب الكلمات التالية ثم اضغط **Space**:
* `باسم` ⮕ بسم الله الرحمن الرحيم
* `صلم` ⮕ صلى الله عليه وسلم
* `تاريخ` ⮕ يدرج تاريخ اليوم تلقائياً.
* `هيد` ⮕ ينشئ ترويسة ملف برمجية احترافية.

### 🔘 أزرار التحكم السريع:
* **[F2]**: لقلب اتجاه الشاشة (من اليمين لليسار والعكس).
* **[F3]**: لتشغيل/إيقاف التدقيق الإملائي العربي.
