# 🌙 ArVim (المحرر العربي الحديث)

**ArVim** هو مشروع مفتوح المصدر يهدف لتطويع محرر النصوص الأسطوري **Vim** لخدمة المستخدم العربي. يوفر الإعدادات اللازمة لجعل الكتابة والبرمجة باللغة العربية تجربة سلسة واحترافية في عام 2026.

## ✨ المميزات الرئيسية:
* **دعم RTL تلقائي:** تهيئة كاملة للكتابة من اليمين إلى اليسار.
* **خرائط الأوامر العربية (Arabic Mapping):** إمكانية إعطاء أوامر Vim (مثل الحفظ، الحذف، التنقل) مباشرة أثناء الكتابة بالعربية دون تبديل لوحة المفاتيح.
* **واجهة معربة:** تعريب شريط الحالة (Statusline) وأوامر الترحيب.
* **خفيف وسريع:** مصمم ليعمل بكفاءة عالية على توزيعات مثل **antiX** والأنظمة الضعيفة.

## 🚀 كيف تبدأ؟
قم بإضافة `source ~/ArVim_Project/core/arvim.vim` إلى ملف `.vimrc` الخاص بك.
